## install_nvm.sh
1. node 버전 확인 <br>
2. 만약 nodejs가 설치되어있지 않다면 <br>
    * nvm 설치 <br>
    * 18.04.6 LTS (Bionic Beaver) 설치 <br>
3. nodejs가 설치되어있으나 v18이 아니라면 <br>
    * v18 설치 후 default를 v18로 변경 <br>
